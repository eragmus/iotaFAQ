  • Basics

  • My transaction has been 'pending' for more than 1 hour. How do I get it to become 'confirmed'?

    These instructions apply to the GUI Desktop wallet. Other wallet may require different steps to be followed.


    1. Click 'HISTORY'.


    2. Click 'Show Bundle' on the 'pending' transaction.


    3. Click 'REATTACH'.


    4. Click 'REBROADCAST'. (optional, usually not required)


    5. Wait for 1 hour.


    6. If still 'pending', repeat steps 1-5 once more.


    IOTA's current Tangle implementation (IOTA is in constant development, so this may change in the future) has a confirmation rate that is ~66% at first attempt.


    So, if a transaction does not confirm within 1 hour, it is necessary to "reattach" (also known as "replay") the transaction one time. Doing so one time increases probability of confirmation from ~66% to ~89%.


    Repeating the process a second time increases the probability from ~89% to ~99.9%.

  • Wallets

  • I want to support IOTA. How can I set up a full node?

    Complete the following steps:


    1. Download the full node software: either GUI, or headless CLI for lower system requirements and better performance.


    2. Get a static IP for your node.


    3. Join the network by adding 7-9 neighbors.


    Try to keep your full node running 24/7 for best results! Either host it on your main computer, a separate computer in your direct possession, or on a suitable hosted VPS.

  • I don't want to set up an IOTA full node. How do I set up a basic IOTA wallet (light node) to send and receive payments?

    There are various options:


    * Download the GUI Desktop wallet, and set up accordingly. This is created by the IOTA Foundation.


    * Download a Mobile wallet for your smartphone: iOS is currently in beta testing, while Android is available. These are created by IOTA's community.


    * Use a Web wallet, accessible via a WebGL2-enabled Internet browser, such as Google Chrome. This is created by IOTA's community.


    More options may become available, as time goes on.
